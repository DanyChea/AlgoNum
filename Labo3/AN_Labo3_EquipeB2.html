Mode d'emploi

Ajout de nouvelles matrices personalisées:

Afin de pouvoir utiliser cet outil avec des matrices personalisées, il est indispensable que les fichiers de matrices soient déposés dans le dossier "json" situé dans le même répertoire que cette page.

De plus, le fichier de matrice doit être au format 'json', carrée et organisé de la manière suivante :

Avec pour n le nombre d'éléments de la matrice carrée.

Pour A, la matrice avec le bon nombre d'éléments, par rapport au n donné.

Pour B, le vecteur solution.

Exemple:

{"n": [3], "A": [3.0,4.0,-1.0,1.0,-1.0,2.0,2.0,3.0,4.0], "B": [23.0,3.0,7.0]}
